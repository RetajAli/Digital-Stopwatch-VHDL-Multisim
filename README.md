# Digital Stopwatch VHDL & Multisim

A Digital Logic Design project that implements a digital stopwatch to count and display minutes and seconds.

The project was built using Multisim digital logic components and includes a bonus VHDL implementation with simulation waveform output.

## Project Features
- Counts seconds and minutes
- Start/Stop control using switches
- Displays time using 7-segment displays
- Uses counters and decoders
- Includes Multisim circuit simulation
- Includes VHDL code and waveform simulation

## Technologies Used
- Multisim
- VHDL
- EDA Playground
- EPWave
- Digital Logic Design

## Components Used
- 7490N Decade Counters
- 7447N BCD to 7-Segment Decoders
- 7-Segment Displays
- 7408N AND Gate
- Function Generator
- Start/Stop Switches

## Repository Structure
- `Documentation/` project report
- `Multisim/` Multisim circuit file
- `VHDL/` VHDL code/screenshots
- `Simulation/` waveform output screenshots
